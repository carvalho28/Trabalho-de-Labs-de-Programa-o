The objective of this project is to implement a program that asks the user 18 whole numbers and keep them in a vector, to later provide shape to calculate some statistics or perform operations on those values. The requested values ​​must be between 6 and 28.

After the values ​​have been requested, a menu is shown to the user who allow you to calculate each of the statistics listed below, exactly in the order placed in this statement. After choosing an option, the The result should be shown on the screen, and the menu should be displayed again.
